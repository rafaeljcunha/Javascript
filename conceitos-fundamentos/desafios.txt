Desafios (Conceitos e fundamentos)

1 - Criar uma função que retorne qual é o tipo do parâmetro que a função recebeu.

2 - Crie um array e adicione os seguintes elementos: banana, abacaxi, morango e uva. Ordene os elementos e imprima o array e em seguida remova o primeiro e o último elemento do array com os métodos próprios para isso e imprima o novo valor do array.

3 - Temos 5 itens em nossa loja, um cliente que saber se existe um determinado item. Crie uma função que receba o nome do item e dentro desta função teremos um array com 5 produtos (da sua escolha), dos quais deveremos filtrar e informar ao cliente se temos ou não este item em estoque.

4 - Um cliente comprou vários itens de uma loja com os seguintes preços: 12.40, 26.70, 120.89, 3.30, 1.25, 1024.00. Crie um array com estes valores, ordene e faça a soma de todos eles para sabermos qual será o valor total da compra.

5 - Converta a string “300” para inteiro e converta o inteiro 1000 para string.

6 - Crie uma função que realize a soma de 3 valores, os valores deve ser recebidos através dos parâmetros da função.

7 - Dado que você possua o seguinte array:

const produtos = [
  {
    titulo: "Notebook",
    preco: 3200.00,
    categoria: "Eletrônicos"
  },
  {
    titulo: "Caderno",
    preco: 10.50,
    categoria: "Papelaria"
  },
  {
    titulo: "Celular",
    preco: 1500.99,
    categoria: "Eletrônicos"
  },
  {
    titulo: "Televisão",
    preco: 2349.99,
    categoria: "Eletrônicos"
  },
  {
    titulo: "Mesa",
    preco: 529.49,
    categoria: "Casa"
  },
  {
    titulo: "Tênis",
    preco: 69.99,
    categoria: "Esporte"
  },
];

Execute as seguintes funções:
1 - Calcular a soma total dos produtos.
2 - Filtrar apenas os produtos da categoria “Eletrônicos”.
3 - Filtrar apenas os produtos com o preço maior ou igual a 1500,00.
4 - Filtrar o produto pelo título “Caderno”
5 - Filtre os produtos que possuem o preço entre 10,00 e 1500,00.
6 - Adicione um novo produto ao array usando o método próprio para isso.
